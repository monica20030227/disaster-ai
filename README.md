# AI 災情可信度分析元件

## 📌 問題
災害期間常出現假訊息、模糊通報，影響決策。

## 💡 解法
本元件透過 AI 分析災情描述，輸出可信度與風險等級。

## ⚙️ 元件類型
API Service Component

## 🔌 API
POST /analyze

### Input
{
  "text": "...",
  "location": "...",
  "time": "..."
}

### Output
{
  "可信度": 0.78,
  "風險等級": "中",
  "不確定性說明": "..."
}
